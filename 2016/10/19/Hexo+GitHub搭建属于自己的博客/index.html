程序员如何建立自己的个人品牌（持续更新）

摘要:
  本文是一篇关于如何在 Windows 下搭建一个独立博客的教程，里面介绍了如何使用 GitHub Pages 和 Hexo 搭建一个属于自己的博客，并配置它的过程。

前言：

  写博客是我程序开发历程中重要的一个环节。大学那会儿最先在CSDN上看别人写的博客，后来就想为何我不自己分享一下所学的东西呢，于是我就尝试在CSDN发表了第一篇技术博客，效果并不怎么好，自己的文字功底都有待提高。就这样发表了几篇博客之后，发现CSDN上博客文章经常显示图片有问题，文本编辑工具也不适应，于是我就从新到微软的OneNote上记录自己的学习笔记，OneNote跨设备同步功能及文本编辑都挺不错的。但是对于一直追崇开源精神的我，OneNote缺少发表文章的功能就满足不了我的需求。刚好那会儿正想搭个属于自己的网站，就利用Octopress + Github的博客框架开始捣鼓起来了自己的网站（注：对Octopress有兴趣的朋友可以移驾到我网易云笔记上的博客教程），但Octopress繁琐的配置和博客迁移使我很头疼，最主要我并不喜欢Octopress上专为黑客设计的黑色风格主题，又找不到合适的主题排版，就这样也差不多抛弃Octopress没什么更新了。刚好前两天看到就职鹅长的一个前端朋友的个人网站，使我眼前一亮，问了才知道原来使用了Hexo + Github的博客框架，遂也决心把自己的博客搬迁到Hexo去。当然搭建的时候必然会踩各种坑，大家有问题可以直接在评论里问，但最好自己先Google百度下。废话不多说，快上车，要开车了。。。

正文:

  Hexo是一款基于Node.js的静态博客框架，并架设在免费的github page平台上，故我们要先把这两个基本工具配置好。
